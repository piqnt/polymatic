## Install

#### NPM
```bash
  npm install polymatic
```

#### CDN - UNPKG
```html
  <script src="//unpkg.com/polymatic@0.0"></script>
```

#### CDN - jsDelivr
```html
  <script src="//cdn.jsdelivr.net/npm/polymatic@0.0"></script>
```
